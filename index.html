What is Charli3

The decentralised oracle network built for Cardano

Charli3 provides verified, tamper-proof external data to smart contracts on Cardano. Its pull oracle model means you request data exactly when you need it, reducing costs and latency for your protocol.

Pull Model
On-Demand Data
Unlike push oracles that constantly update onchain, pull oracles only fetch data when your smart contract requests it. You pay per request, not per block.
Multi Node Proof
Cryptographically Verified
Every oracle response is signed by multiple independent nodes. Your validator verifies the aggregated proof onchain before acting on the data.
Open Source
MIT Licensed Infrastructure
The full pull oracle stack, SDK, contracts, and node software is open source and freely available. Anyone can integrate, fork, or extend it.
Use Cases

What you can build with
Charli3 pull oracles

DeFi
Price-Triggered Liquidations
Auto liquidate undercollateralised positions the moment onchain price data confirms a threshold is breached. No keepers, no oracles polling on a schedule.
Settlements
Parametric Insurance
Release insurance payouts the instant oracle data confirms a qualifying event: a weather threshold, a sports outcome, or a supply chain confirmation.
Synthetics
Dynamic Asset Minting
Mint synthetic assets against verified collateral ratios. Oracle data at mint time ensures every token is backed by real time market data.
